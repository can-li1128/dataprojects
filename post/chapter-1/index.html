Project 1: Iris Identification

January 15, 2021

Overview

Applying machine learning method, this project takes sepal and petal measurement data from the famous (Fisher’s or Anderson’s) iris data set to predict species, versicolor or virginica.

A continuous sequence of cutoff points were used to find the best measurement that produces highest accuracy on training data; then the same measurement was applied to test data to evaluate the algorithm.

Two specific features are proved to be most useful in determining iris species.

Load the following libraries and dataset

    library(tidyverse)
    library(caret)
    data(iris)

Generate trainging and test index:

    test_index <- createDataPartition(y, times = 1, p = 0.5, list = FALSE)

Test out prediction accuracy:

Create a sequence of cutoffs within 2 sd away from mean with increments of 0.1 cm; the following figure shows the best cutoff point to predict with the highest accuracy. After this method was applied to all four measurements, petal length seems most predictive (with accuracy of 96% at 4.7 cm cutoff point) on training data.

    accuracy <- map_dbl(cutoff, function(x){
      y_hat <- ifelse(train$Petal.Length > x, "virginica", "versicolor") %>%
        factor(levels = levels(test$Species))
      mean(y_hat == train$Species)
    })

## [1] 0.96

Further analysis

However, when 4.7 cm was applied on test data for evaluation, accuracy was lower, while petal width at optimal cutoff point of 1.6 cm gives higher accuracy. This indicates single predictor is not the best choice. Algorithm was adjusted to combine two features to predict virginica if either petal length is greater than 4.7 cm OR petal width greater than 1.6 cm and versicolor otherwise. This achieved highest Balanced Accuracy.

    confusionMatrix(y_hat, test$Species)
