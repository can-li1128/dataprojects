Project 3: Text Analysis with Barry's Blog

February 1, 2021

Overview

Text analysis techniques generate numeric summaries of text data and apply visualization to present these summaries. This a text analysis project with Barry Pomeroy’s blog entry called My Brother’s Dinner with Andre.

I used web scraping to harvest the blog entry, transformed the text into word units, then used sentiments lexicons AFINN, bing, and nrc to analyze the text. Steps are detailed below.

Part 1: Text Analysis

Load libraries

Use the following libraries. rvest package is for web scraping, and tidytext package is used to convert free text into tidy table.

      library(tidyverse)
      library(rvest)
      library(tidytext)
Read in the blog entry
    url <- "http://barrypomeroy.com/blog/my-brothers-dinner-with-andre/"
    dinner <- read_html(url)
Select html nodes needed

Use selectorGadgets to find the html node that contain text. In this case, the web page is very clean since all the text are contained in p nodes. Once paragraph node set are harvested, they are transformed into a “character” vector.

    para_nodes <- html_nodes(dinner, "p")
    para <- para_nodes %>% html_text()
Transform text and extract words

Vector is transformed into a table, each observation contains a paragraph. Use unnest_tokens() from rvest package to create a tidy table.

    df_para <- tibble::enframe(para)
    wd <- df_para %>% unnest_tokens(word, value)
    head(wd)

Part 2: Sentiment Analysis

  • Using a dictionary based approach to analyze the sentiment of a text, opinions and emotions, is ONE of the methods for sentiment analysis.
  • The tidytext package provides several sentiment lexicons. Three general-purpose lexicons are AFINN, bing, and nrc. In this report, I only use nrc lexicon as an example. There are some slight differences in comparison with the other two dictionaries.
  • These lexicons are bases for sentiment analysis with ample flexibility to have words added or withdrawn according to the nature of the text, words, and project.
nrc negative

Use nrc lexicon to identify negative words

    nrc_neg <- get_sentiments("nrc") %>% filter(sentiment == "negative")
