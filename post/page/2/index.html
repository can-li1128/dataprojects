Projects

I started learning data science on the August long weekend of 2020 as my pandemic project. As soon, I found myself immersed in various projects, excited about what can be discovered and understood.

In documenting these selected projects, I demonstrate various data science methodologies used in these projects, from simple computation to statistical methods to machine learning.

Project background is described in overview; although not every step is detailed, R packages, data set, and important code chucks used in each projects are listed. These documents are produced in R Markdown reports.

January 15, 2021

Project 1: Iris Identification

Applying machine learning method, this project takes sepal and petal measurement data from the famous (Fisher’s or Anderson’s) iris data set to predict species, versicolor or virginica. A continuous sequence of cutoff points were used to find the best measurement that produces highest accuracy on training data; then the same measurement was applied to test data to evaluate the algorithm. Two specific features are proved to be most useful in determining iris species.
